#A 6 x 6 tic tac toe game#
###Author: Abhay Mittal###
I did this project in my sophomore year.

This game is built by implementing a minimax algorithm using alpha beta pruning. The winning combination consisted of 4 adjacent moves placed together. Human player is marked yellow and CPU is red. 

###Steps to execute###
	1. `javac GameMain.java`
	2. `java GameMain`
